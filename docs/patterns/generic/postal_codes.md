# Patterns: Postal Codes

## Postal Codes

| Pattern Name | Regex |
|---|---|
| US ZIP+4 Code | `\b\d{5}-\d{4}\b` |
| UK Postcode | `\b[A-Z]{1,2}\d[A-Z0-9]?\s?\d[A-Z]{2}\b` |
| Canada Postal Code | `\b[A-Z]\d[A-Z]\s?\d[A-Z]\d\b` |
| Japan Postal Code | `\b\d{3}-\d{4}\b` |
| Brazil CEP | `\b\d{5}-\d{3}\b` |
